# 功能規格文件

這份文件描述目前網站的產品功能與使用者可見行為，方便後續判斷需求是否已完成、UI 行為是否符合預期。

## 導覽頁面

網站目前主要分成以下頁面：

- 專輯頁：瀏覽與搜尋塞壬唱片專輯。
- 幹員圖鑑：瀏覽角色列表、篩選角色、查看角色詳情。
- 招募卡：製作角色招募卡。
- 我的清單：登入後管理我的最愛、歌單、角色清單。
- 登入/註冊：使用唯一登入 Key 與密碼建立使用者。

## 音樂播放

### 專輯播放

- 使用者可從專輯詳情選擇歌曲播放。
- 播放器會顯示封面、歌曲名稱、歌手、進度、音量、歌詞與歌曲資訊。
- 從專輯播放時，播放 queue 以該專輯歌曲為範圍。

### 歌單播放

- 使用者可在我的歌單中查看歌曲列表。
- 歌單內歌曲不以整列點擊播放，而是使用獨立播放按鈕。
- 從歌單播放指定歌曲時，會從該歌曲位置開始播放。
- 在歌單來源播放時，隨機播放範圍限制在該歌單內。

### 播放模式

同一個播放模式按鈕依序切換：

1. 循環播放
2. 單曲循環
3. 隨機播放

## 歌詞與翻譯

- 歌詞翻譯預設關閉。
- 切換歌曲後，翻譯狀態會回到關閉。
- 中文介面下，英文歌詞應翻譯成中文，中文歌詞不需要再翻譯成英文。
- 翻譯請求由 Worker 後端處理並快取。

## 分享連結

- 歌曲可產生分享連結。
- 分享連結支援 Open Graph 預覽。
- 分享頁由 Worker 回傳對應 HTML metadata。

## 使用者系統

### 登入與註冊

- 使用者不一定要登入。
- 未登入仍可瀏覽專輯、播放歌曲、查看角色與使用招募卡。
- 登入 Key 必須唯一。
- 登入後可同步收藏、歌單與角色清單。

### 我的最愛

- 我的最愛按下後直接加入或移除。
- 操作完成後顯示通知。

### 我的歌單

- 使用者可建立、重新命名、刪除歌單。
- 可加入或移除歌曲。
- 可播放全部歌曲。
- 可查看目前歌曲已加入哪些歌單。
- 若尚未建立歌單，初次加入時應引導建立。

### 角色清單

- 使用者可建立自己的角色清單。
- 角色清單中的角色可點擊查看角色詳情。
- 角色圖片若載入失敗，使用 fallback icon，不顯示破圖。

## 角色功能

- 角色列表支援搜尋、稀有度、職業、陣營等篩選。
- 角色詳情包含基本資訊、立繪、天賦/特性、技能、模組、材料、語音或檔案等資料。
- 角色詳情頁導覽提供回角色清單與回首頁。

## 非功能規格

- 手機版應避免不必要的大量預載。
- 外部圖片來源不穩時，應由 Worker cache/fallback 降低失敗率。
- 使用者資料 API 需要處理 CORS。
- 高風險資料如 Supabase service role key 不可提交到 Git。
